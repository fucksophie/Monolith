Things you'll need

  1. git (downloadable from https://git-scm.com/)
  2. a C complier such as (Mingw?) GCC/Clang
  3. File browser
  4. Sensible file editor (Visual Studio Code, or notepad would work)

Instructions

  1. Clone the repository (git clone https://github.com/UnknownShadow200/ClassiCube)
  2. With a text editor, navigate to classicube/src/Constants.h
  3. Replace the #define strings with something like this (replace localhost to your own server) 
    #define SKINS_SERVER    "http://localhost/skin"
#define UPDATES_SERVER  "http://cs.classicube.net/client"
#define SERVICES_SERVER "http://localhost/api"
#define RESOURCE_SERVER "http://static.classicube.net"
#define REGISTERNEW_URL "http://localhost/register"
  4. Compile based on the instructions in the readme of the repository
  5. Make a new folder called classicubetesting
  6. Move the classicube/src/ClassiCube executable to classicubetesting/ClassiCube
  7. Go into classicubetesting and launch it via ./ClassiCube
    8. (linux)
